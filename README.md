# 心晨V1.0.0使用说明书
## 一、软件介绍
“心晨” 是一款专注于情绪管理与心理关怀的手机应用，开发旨在帮助用户精准觉察自身情绪状态、科学调节负面情绪，通过多样化功能实现心理自我关怀与成长。

软件核心整合四大核心功能板块：一是全面的心情记录系统，支持情绪量化记录、历史查询及情绪曲线追踪，助力用户洞察情绪变化规律；二是多元情绪调节系统，涵盖情绪调节包、心理测评、心理知识科普等丰富资源，兼顾实用调节工具与成长型知识储备；三是特色时光邮件功能，支持用户与未来自我跨时空对话，传递成长期许；四是趣味心情花园互动功能，通过任务激励机制引导用户积极参与情绪管理，在轻松种植中滋养心态，全方位为用户心理健康保驾护航。

## 二、主要功能介绍
### 1．心情记录模块
#### （1）首页
点击底部导航栏的“首页”，可以进入首页页面。

滑动首页的滚动条，可以量化选择此刻的心情值，点击“记录下来！”即可进入“添加记录页面”详细记录该条心情。

在“添加记录页面”中，可以自定义一个此刻的状态、输入此刻想表达的内容、添加图片，点击保存后该条记录将保存在当天。

### 2、调节心情模块
点击底部导航栏的“调节”，进入调节心情模块



#### (1) 情绪调节包
点击上方的“情绪调节”，可以看到一个带有丰富功能的情绪调节包，里面有14个调节情绪的小功能，用户可以选择里面适合的调节功能调节情绪。
用户可以收藏喜欢的调节功能，然后在“首页”的“我的调节包”悬浮球快捷打开。

#### 2、 心理测评、心理知识、哲学知识、情绪课堂
点击上方的“心理测评”，用户可以做一些热门的有趣的心理测评量表（比如MBTI、霍兰德职业测试等等）。

此外，点击上方其他选项，还有心理知识、哲学知识、情绪课堂等诸多有助于调节心情的知识科普。

本页面提供的心理测评(含MBTI、大五人格、焦虑自评等)均为自我认知参考工具，仅用于辅助个人了解自身心理特质、兴趣倾向或情绪状态，不构成专业心理诊断、医疗建议或治疗方案。

### 3、时光邮件
点击底部导航栏的“邮件”页面，进入时光邮件模块。

时光邮件是“心晨”的特色功能之一，用户可以点击“写邮件”按钮写一封邮件给未来的自己，时间到了之后该邮件会在收件箱中出现，从而实现“跨时空对话”。

### 四、 心情花园
点击底部导航栏的“心园”页面，进入心情花园页面。

心情花园也是“心晨”的特色功能之一。用户可以用“开心水”来种植“开心花”，用户点击页面中的“浇水”，可以使开心花植株从发芽开始逐渐成长，直至。用户可以随时种植一盆新的花。

用户在“添加记录页面”的时候若保存记录，可获得1点“开心水”，在调节页面中的“情绪调节包”中的“心理暗示”每添加一条心理暗示，可获得3点“开心水”。在“时光邮件”中每发送一封邮件，可获得5点“开心水”。
